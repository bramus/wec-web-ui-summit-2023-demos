# WEC Web UI Summit 2023

Source files for the WEC Web UI Summit 2023 demos.

## Installation

-   Clone this Repo
-   Install dependencies using NPM

    ```bash
    npm i
    ```

## Running the project

-   Start the local server

    ```bash
    npm start
    ```

-   Visit [http://localhost:3000/](http://localhost:3000/)
